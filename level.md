## 🧩 Node System

Setiap node memberikan XP tertentu:

| Node      | XP         |
| --------- | ---------- |
| 1         | 150        |
| 2         | 100        |
| 3         | 200        |
| 4         | 200        |
| 5         | 150        |
| **Total** | **800 XP** |

---

## 📝 Quiz System

XP didapat dari hasil kuis:

| Keterangan            | Nilai      |
| --------------------- | ---------- |
| Jumlah Soal           | 20         |
| XP per Benar          | 10         |
| XP per Salah          | 0          |
| **Total XP Maksimal** | **200 XP** |

---

## 🔥 Summary

* Total XP dari Node: **800 XP**
* Total XP dari Quiz: **200 XP**
* **Total Maksimal XP: 1000 XP (Level 10)**
